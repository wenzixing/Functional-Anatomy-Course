# 第17章：运动功能测试

## 学习目标
- 掌握运动功能测试的基本原理
- 理解常见功能性测试方法
- 认识测试结果的解释
- 理解功能评估在康复中的应用

## 17.1 运动功能测试的原则

### 测试的目的
- **基线评估**：建立初始状态
- **进展评估**：追踪改善过程
- **预后评估**：预测康复结果
- **返回运动**：判断返回运动能力

### 良好测试的特点
- **信度**：重复测试结果一致
- **效度**：真正衡量要评估的内容
- **敏感性**：能检测到细微改变
- **可行性**：在临床环境中可行

### 测试原则
1. **安全第一**：保护患者安全
2. **清晰说明**：让患者理解步骤
3. **标准化**：保证一致的测试条件
4. **记录详细**：便于比较进展

## 17.2 上肢功能测试

### 握力测试

**工具**：握力计

**标准操作**：
1. 坐位，肘部90°屈曲，前臂中立位
2. 握力计置于手中，用力握
3. 维持3秒，记录数值
4. 两手各测3次，取平均值

**正常范围**：
- 男性：>40kg
- 女性：>25kg

**临床意义**：
- 评估手部整体力量
- 快速筛查神经肌肉问题
- 追踪康复进展

### 九孔钉板测试（Nine Hole Peg Test）

**工具**：九孔板、9颗钉子

**标准操作**：
1. 患者坐在桌前
2. 将钉子逐个放入孔中
3. 完成后逐个取出
4. 记录总时间

**正常范围**：<25秒（健康成人）

**临床意义**：
- 评估上肢精细运动
- 神经系统康复评估
- 中风、帕金森病评估

### 肩关节活动度测试（ROM）

**工具**：量角器或关节角度计

**主要测试**：
- 前屈：正常180°
- 外展：正常180°
- 内旋：正常75-90°
- 外旋：正常75-90°

**测试方法**：
1. 被测者站立或坐位
2. 执行被动运动到末端感觉
3. 用量角器测量角度
4. 记录数值

**异常表现**：
- 活动度减少：提示肌肉紧张或关节囊紧张
- 疼痛限制：提示结构病变
- 代偿运动：提示稳定性问题

### 肘部测试

**肘部活动度**：
- 屈曲：130-140°
- 伸展：通常0°，部分人可伸展

**旋转活动度**：
- 旋后：85-90°
- 旋前：80-85°

## 17.3 下肢功能测试

### 单腿站立测试

**标准操作**：
1. 患者赤足站立
2. 单腿站立，对侧腿屈曲
3. 闭眼或睁眼站立
4. 记录持续时间

**正常范围**：
- 睁眼：>30秒
- 闭眼：>25秒

**临床意义**：
- 评估下肢稳定性
- 评估本体感受
- 跌倒风险评估

**异常表现**：
- 下肢晃动：肌肉无力
- 迅速跌落：严重不稳定
- 闭眼差异大：本体感受缺陷

### 30秒坐立测试（30-Second Sit-to-Stand Test）

**标准操作**：
1. 患者坐在椅子上（坐垫高度约43cm）
2. 双脚分开肩宽
3. 双臂交叉于胸前
4. 尽可能多地完成站起-坐下循环
5. 计数30秒内完成的次数

**正常范围**：
- 60-69岁：12-16次
- 70-79岁：11-14次
- 80-89岁：8-12次

**临床意义**：
- 评估下肢力量和耐力
- 跌倒风险评估
- 运动能力评估

### Y字平衡测试（Y Balance Test）

**工具**：特殊的Y形支架或标记

**标准操作**：
1. 单腿站立在中心点
2. 对侧腿向三个方向伸展（前、后外侧、外侧）
3. 脚尖触及最远点后返回
4. 记录三个方向的最大距离
5. 计算对称指数 = 无损伤腿距离 / 有损伤腿距离 × 100%

**正常范围**：对称指数>90%

**临床意义**：
- 评估动态平衡
- 下肢功能评估
- 运动员伤害预防

### 膝关节活动度测试

**膝关节活动度**：
- 伸展：0°（有些人可过伸5-10°）
- 屈曲：130-140°

**Q角测量**：
- 定义：髂前上棘到膝中心、膝中心到胫骨粗隆的角度
- 正常范围：男性<20°，女性<25°
- >20°：膝内扣风险增加

## 17.4 脊柱功能测试

### 指地测试（Fingertip-to-Floor Test）

**标准操作**：
1. 双脚并立，膝关节伸展
2. 向前弯腰，双臂下垂
3. 手指尽可能接近地面
4. 测量手指与地面的距离

**正常范围**：
- 健康年轻人：手指接触地面或更近
- 中年人：10-20cm距离
- 老年人：20-30cm距离

**临床意义**：
- 脊柱屈曲灵活性
- 腘绳肌紧张程度
- 腰椎活动度

### 托马斯试验（Thomas Test）

**目的**：评估髋屈肌（腰大肌）的紧张程度

**标准操作**：
1. 患者仰卧在检查床边，臀部在床边
2. 健康侧膝屈曲，抱向胸部（消除腰椎前凸）
3. 观察患侧下肢
4. 如果患侧大腿抬起：提示髋屈肌紧张

**结果解释**：
- 大腿水平或抬起：髋屈肌紧张
- 大腿下垂：髋屈肌正常

### 奥伯试验（Ober Test）

**目的**：评估髋外展肌（特别是IT带）的紧张程度

**标准操作**：
1. 患者侧卧，下侧膝屈曲提供稳定
2. 上侧髋伸展、外展，膝屈曲
3. 观察上侧膝是否下落到水平线以下
4. 若膝保持在水平线以上：提示IT带紧张

**结果解释**：
- 膝保持高位：IT带紧张
- 膝下落：IT带正常

## 17.5 功能性活动测试

### 上楼速度测试（Stair Climbing）

**标准操作**：
1. 患者以正常速度爬楼梯
2. 计时上升和下降时间
3. 观察姿态和代偿模式

**观察重点**：
- 运动顺序：是否正确
- 速度：是否减慢
- 姿态：是否存在代偿
- 安全性：是否需要扶手

### 前弓步测试（Lunge Test）

**标准操作**：
1. 双足站立，肩宽距离
2. 一条腿向前跨，前膝屈曲至90°
3. 后膝接近地面
4. 保持平衡3秒，返回
5. 两侧各重复

**评估内容**：
- 平衡能力
- 下肢力量
- 代偿模式

### 步态速度测试（Gait Speed Test）

**标准操作**：
1. 标记10米直线
2. 患者以正常速度行走
3. 计时通过10米的时间
4. 计算步速（m/s）

**正常范围**：
- 年轻人：1.4-1.6 m/s
- 老年人：0.9-1.2 m/s
- <0.6 m/s：严重功能受限

## 17.6 功能评估的综合应用

### 评估流程

1. **初始评估**：建立基线
2. **进展评估**：每2-4周重复
3. **最终评估**：康复结束
4. **随访评估**：长期监测

### 结果解释

**改善**：表现达到或超过正常值
**部分改善**：表现改善但未达正常值
**无改善或恶化**：需要调整计划

## 要点总结

✓ 功能测试应遵循**标准化、安全、可重复**的原则
✓ 不同测试评估**不同系统和功能**
✓ **系列测试**而非单一测试更能全面评估
