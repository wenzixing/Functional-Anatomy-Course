# 第5章：肩部解剖学与功能

## 学习目标
- 掌握肩关节的解剖特点
- 理解肩部肌肉的功能分布
- 认识肩关节的活动范围
- 理解肩部常见功能障碍

## 5.1 肩部骨骼结构

### 肩关节的组成
- **肱骨头**：球形，直径大
- **肩胛骨盂**：浅窝形，面积小
- 特点：活动度大，稳定性差

### 相关骨骼结构
| 骨骼 | 解剖位置 | 功能 |
|------|---------|------|
| 肩胛骨 | 胸壁后方 | 提供活动平台 |
| 锁骨 | 胸骨-肩峰 | 维持肩部距离 |
| 肱骨 | 上臂 | 提供活动杆臂 |

### 肩胛骨的位置标志
- **肩峰**：最外侧突出点
- **肩胛棘**：后方隆起
- **下角**：下方转折点
- **上角**：内侧上方

## 5.2 肩部关节

### 肩关节（肩盂肱关节）
- **关节面**：肱骨头与肩胛盂
- **结构**：球窝关节，最灵活
- **活动范围**：前屈180°，外展180°

### 肩锁关节（AC关节）
- **功能**：允许锁骨旋转
- **活动幅度**：小，25-30°

### 胸锁关节（SC关节）
- **功能**：锁骨与胸骨的连接
- **作用**：协调上肢运动

### 肩胛骨与胸壁的关系
- **滑动面**：肩胛骨在胸壁上滑动
- **韧带加固**：不是真正的关节
- **肌肉驱动**：前锯肌、肩胛提肌等

## 5.3 肩部主要肌肉

### 旋转袖肌肉（Rotator Cuff）

| 肌肉 | 起点 | 止点 | 功能 |
|------|------|------|------|
| 冈上肌 | 冈上窝 | 大结节上 | 外展 |
| 冈下肌 | 冈下窝 | 大结节下 | 外旋 |
| 小圆肌 | 腋边 | 大结节下 | 外旋 |
| 肩胛下肌 | 肩胛下面 | 小结节 | 内旋 |

### 肩部主要肌肉功能

**上举运动**
- 0-15°：冈上肌
- 15-90°：三角肌前部 + 冈上肌
- 90-180°：三角肌中部 + 肩胛骨上回旋

**旋转运动**
- 外旋：冈下肌、小圆肌
- 内旋：肩胛下肌、胸大肌

## 5.4 肩胛骨的运动

### 肩胛骨的运动类型
```
上提 ↑     下降 ↓
      ↙ ↖
    内收      外展（前锯）
      ↗ ↘
  上回旋      下回旋
```

### 肌肉控制

| 运动 | 主要肌肉 |
|------|---------|
| 上提 | 肩胛提肌 |
| 下降 | 下部斜方肌 |
| 外展 | 前锯肌 |
| 内收 | 菱形肌 |
| 上回旋 | 上部斜方肌 + 下部斜方肌 |
| 下回旋 | 肩胛下肌 + 菱形肌 |

## 5.5 肩关节的活动控制与范围（ROM）

肩关节的运动是典型的多轴运动，其复杂性在于**盂肱关节**与**肩胛胸壁单位**的精密协作。

### 5.5.1 三面三轴的运动定义

| 运动类型 | 发生平面 | 转动轴 | 正常范围 | 主要原动肌 |
| :--- | :--- | :--- | :--- | :--- |
| **前屈/后伸** | **矢状面** | **冠状轴** | 前屈180° / 后伸60° | 三角肌前部、胸大肌 / 背阔肌、大圆肌 |
| **外展/内收** | **冠状面** | **矢状轴** | 外展180° / 内收15° | 三角肌中部、冈上肌 / 胸大肌、背阔肌 |
| **旋内/旋外** | **横断面** | **垂直轴** | 旋内105° / 旋外90° | 肩胛下肌、胸大肌 / 冈下肌、小圆肌 |

### 5.5.2 肩胛胸壁节律 (Scapulohumeral Rhythm)
肩部上举运动并非单纯由盂肱关节完成：
- **比例关系**：在上举全过程中，盂肱关节与肩胛骨旋转的比例约为 **2:1**。
- **功能意义**：肩胛骨的上回旋不仅增加了运动范围，还维持了肱骨头在关节盂中心的力量平衡，防止撞击。

### 5.5.3 运动学代偿现象
当肩部出现功能障碍时，人体常通过以下方式进行代偿，这在评估中需重点关注：
- **脊柱侧屈**：试图通过躯干倾斜来辅助完成冠状面内的外展。
- **脊柱过伸**：在矢状面内前屈受限时，通过腰椎前凸增加来“借力”。
- **耸肩**：利用肩胛提肌过度代偿受损的旋转袖功能。

## 5.6 肩部常见功能障碍

### 肩关节撞击症候群
- **原因**：肩峰下空间狭窄
- **症状**：特定角度疼痛
- **治疗**：肩胛骨稳定性训练

### 肩关节不稳定
- **原因**：旋转袖虚弱或关节囊松弛
- **症状**：肩关节脱位或半脱位
- **治疗**：加强旋转袖肌肉

### 冻结肩（粘连性关节囊炎）
- **原因**：关节囊炎症和粘连
- **症状**：活动范围严重受限
- **分期**：冻结期 → 缓解期 → 恢复期

## 要点总结

✓ 肩关节是**活动度最大**但**稳定性最差**的关节
✓ **旋转袖肌肉**对肩关节稳定至关重要
✓ **肩胛骨**运动与**肩关节**运动协调进行
