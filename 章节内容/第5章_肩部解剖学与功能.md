# 第5章：肩部解剖学与功能

## 学习目标
- 掌握肩关节的解剖特点
- 理解肩部肌肉的功能分布
- 认识肩关节的活动范围
- 理解肩部常见功能障碍

## 5.1 肩部骨骼结构

### 肩关节的组成
- **肱骨头**：球形，直径大
- **肩胛骨盂**：浅窝形，面积小
- 特点：活动度大，稳定性差

### 相关骨骼结构
| 骨骼 | 解剖位置 | 功能 |
|------|---------|------|
| 肩胛骨 | 胸壁后方 | 提供活动平台 |
| 锁骨 | 胸骨-肩峰 | 维持肩部距离 |
| 肱骨 | 上臂 | 提供活动杆臂 |

### 肩胛骨的位置标志
- **肩峰**：最外侧突出点
- **肩胛棘**：后方隆起
- **下角**：下方转折点
- **上角**：内侧上方

## 5.2 肩部关节

### 肩关节（肩盂肱关节）
- **关节面**：肱骨头与肩胛盂
- **结构**：球窝关节，最灵活
- **活动范围**：前屈180°，外展180°

### 肩锁关节（AC关节）
- **功能**：允许锁骨旋转
- **活动幅度**：小，25-30°

### 胸锁关节（SC关节）
- **功能**：锁骨与胸骨的连接
- **作用**：协调上肢运动

### 肩胛骨与胸壁的关系
- **滑动面**：肩胛骨在胸壁上滑动
- **韧带加固**：不是真正的关节
- **肌肉驱动**：前锯肌、肩胛提肌等

## 5.3 肩部主要肌肉

### 旋转袖肌肉（Rotator Cuff）

| 肌肉 | 起点 | 止点 | 功能 |
|------|------|------|------|
| 冈上肌 | 冈上窝 | 大结节上 | 外展 |
| 冈下肌 | 冈下窝 | 大结节下 | 外旋 |
| 小圆肌 | 腋边 | 大结节下 | 外旋 |
| 肩胛下肌 | 肩胛下面 | 小结节 | 内旋 |

### 肩部主要肌肉功能

**上举运动**
- 0-15°：冈上肌
- 15-90°：三角肌前部 + 冈上肌
- 90-180°：三角肌中部 + 肩胛骨上回旋

**旋转运动**
- 外旋：冈下肌、小圆肌
- 内旋：肩胛下肌、胸大肌

## 5.4 肩胛骨的运动

### 肩胛骨的运动类型
```
上提 ↑     下降 ↓
      ↙ ↖
    内收      外展（前锯）
      ↗ ↘
  上回旋      下回旋
```

### 肌肉控制

| 运动 | 主要肌肉 |
|------|---------|
| 上提 | 肩胛提肌 |
| 下降 | 下部斜方肌 |
| 外展 | 前锯肌 |
| 内收 | 菱形肌 |
| 上回旋 | 上部斜方肌 + 下部斜方肌 |
| 下回旋 | 肩胛下肌 + 菱形肌 |

## 5.5 肩关节的活动范围（ROM）

### 三个平面的运动
- **矢状面**：前屈180°，后伸60°
- **额状面**：外展180°，内收15°
- **横断面**：外旋90°，内旋105°

### 关节活动的代偿
- 肩胛骨参与（肩胛胸壁节律）
- 脊柱参与（侧屈、旋转）
- 脚踏参与（转身时的稳定）

## 5.6 肩部常见功能障碍

### 肩关节撞击症候群
- **原因**：肩峰下空间狭窄
- **症状**：特定角度疼痛
- **治疗**：肩胛骨稳定性训练

### 肩关节不稳定
- **原因**：旋转袖虚弱或关节囊松弛
- **症状**：肩关节脱位或半脱位
- **治疗**：加强旋转袖肌肉

### 冻结肩（粘连性关节囊炎）
- **原因**：关节囊炎症和粘连
- **症状**：活动范围严重受限
- **分期**：冻结期 → 缓解期 → 恢复期

## 要点总结

✓ 肩关节是**活动度最大**但**稳定性最差**的关节
✓ **旋转袖肌肉**对肩关节稳定至关重要
✓ **肩胛骨**运动与**肩关节**运动协调进行
