# 第8章：上肢运动链

## 学习目标
- 理解上肢作为整体运动系统
- 掌握上肢各关节的协调原理
- 认识运动链中的代偿模式
- 理解功能性上肢活动

## 8.1 运动链的基本概念

### 什么是运动链？
- **定义**：多个关节和肌肉系统协调工作的整体
- **特点**：一个环节的问题影响整个链
- **类型**：开放链（末端自由）、闭合链（末端固定）

### 上肢的链式特点
```
肩关节 → 肘关节 → 腕关节 → 手指
  |        |        |        |
肩胛骨    旋转    支点    操作
```

## 8.2 上肢运动的动力学基础

### 近端到远端的力量传递
- **起点**：躯干和肩胛骨的稳定
- **中间**：肘关节的力量传递
- **终点**：腕手的精细控制

### 动作的三个阶段

**准备阶段**
- 肩胛骨位置建立
- 肘部预定位
- 腕部准备

**加速阶段**
- 肩部启动
- 肘部开展
- 力量逐级传递

**减速阶段**
- 肌肉离心收缩
- 控制运动幅度
- 准备下一步

## 8.3 典型上肢功能运动

### 举臂运动（Reaching）
```
0-15°    → 冈上肌主导
15-90°   → 三角肌中部 + 肩胛骨外展
90-180°  → 肩胛骨上回旋 + 脊柱代偿
```

**配合肌肉**：前锯肌、斜方肌、菱形肌

### 投掷运动（Throwing）
**准备期**
- 肩胛骨内收
- 肩关节外旋最大
- 肘部屈曲90°

**加速期**
- 肩胛骨外展
- 肩关节内旋
- 肘部迅速伸展
- 腕部鞭状运动

**减速期**
- 旋转袖肌肉离心收缩
- 肘部伸展肌肉制动
- 腕部屈肌制动

### 握拳与操作运动

**握拳建立**
- 腕部伸展（提升握力）
- 指屈肌激活
- 手部内在肌精细控制

**精细操作**
- 腕部中立位置
- 手指独立运动
- 触觉反馈指导

## 8.4 上肢运动的肌肉协调

### 协同作用的例子

**肩关节前屈运动**
```
原动肌：    肩前部三角肌、胸大肌前部
协同肌：    肱二头肌（屈肘时）
稳定肌：    脊柱竖肌（维持脊柱）、腹肌（控制躯干）
对抗肌：    背阔肌（后伸时）
```

### 上肢运动链的肌肉招募顺序
1. **躯干肌肉**：建立稳定基础
2. **肩部肌肉**：定位和启动
3. **肘部肌肉**：力量传递
4. **腕手肌肉**：精细控制

## 8.5 上肢运动的速度与力量

### 速度与力量的平衡
- **力量运动**：大幅度、低速度、高张力（举重）
- **速度运动**：小幅度、高速度、协调（投掷）
- **耐力运动**：中等幅度、持续重复（游泳）

### 功率的生成
$$P = \frac{\text{力} \times \text{距离}}{\text{时间}} = \text{力} \times \text{速度}$$

### 上肢功率的优化
- 增加肌肉力量（力量训练）
- 增加运动速度（速度训练）
- 改善协调效率（技能训练）

## 8.6 代偿运动模式

### 常见代偿模式

**肩峰下撞击的代偿**
- 肩胛骨上回旋不足
- 脊柱侧屈补偿
- 躯干旋转参与

**握力弱的代偿**
- 腕部过屈补偿
- 肘部屈曲增加
- 躯干转向辅助

### 识别代偿的重要性
- **早期识别**：防止进一步损伤
- **针对根本原因**：治疗根本问题
- **恢复正常模式**：防止慢性问题

## 8.7 上肢功能的评估

### 功能性测试

| 测试 | 目的 | 正常标准 |
|------|------|---------|
| 肩关节ROM | 评估活动度 | 前屈180°等 |
| 握力测试 | 评估力量 | 男性>40kg |
| 九孔钉板测试 | 评估精细运动 | <25秒 |
| 肩梯形测试 | 评估运动控制 | 无代偿 |

### 综合评估框架
1. **被动活动度**：关节潜能
2. **主动活动度**：肌肉能力
3. **肌力测试**：具体肌肉
4. **功能活动**：实际应用能力

## 要点总结

✓ 上肢运动是**多关节、多肌肉**的**协调链式运动**
✓ **近端稳定**是**远端活动**的基础
✓ 识别和纠正**代偿模式**对康复至关重要
