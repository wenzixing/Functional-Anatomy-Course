# 第14章：脊柱运动功能

## 学习目标
- 理解脊柱各个区段的运动特性
- 掌握脊柱运动的力学原理
- 认识脊柱运动与躯干肌肉的关系
- 理解脊柱功能性运动

## 14.1 脊柱运动的运动学概览

脊柱的运动是由多个椎间关节微小运动累积而成的宏观表现，不同区段的运动潜力受关节突关节平面朝向的引导。

### 14.1.1 脊柱运动汇总表

| 运动方向 | 发生平面 | 转动轴 | 灵活性最高的区段 | 主要限制因素 |
| :--- | :--- | :--- | :--- | :--- |
| **屈曲 (Flexion)** | **矢状面** | **冠状轴** | 腰椎 (L1-L5) | 后方韧带、胸廓、椎间盘后部 |
| **伸展 (Extension)** | **矢状面** | **冠状轴** | 颈椎 (C-spine) | 关节突撞击、前纵韧带 |
| **侧屈 (L-Flexion)** | **冠状面** | **矢状轴** | 颈椎 / 腰椎 | 横突间韧带、肋骨 |
| **旋转 (Rotation)** | **横断面** | **垂直轴** | **寰枢关节 (C1-C2)** / **胸椎** | 椎间盘纤维环、棘突间距 |

### 14.1.2 各区段运动角度参考 (总范围)

| 区段 | 屈/伸 | 左右侧屈 | 左右旋转 |
| :--- | :--- | :--- | :--- |
| **颈椎** | 90° (45/45) | 90° (45/45) | 160° (80/80) |
| **胸椎** | 70-95° | 40-80° | 60-80° |
| **腰椎** | 90-105° (60/35) | 40-60° | 10-20° |

## 14.2 脊柱运动的力学原理

### 脊柱弯曲的生物力学

**椎间盘的应力分布**
```
中立位置 → 压力均匀分布
        ↓
前弯时 → 椎间盘前部压力增加
        ↓
后弯时 → 椎间盘后部压力增加
```

### 脊柱稳定性与灵活性的平衡
- **稳定**：椎体大、韧带强、肌肉发达
- **灵活**：椎间盘厚、关节面斜向、活动度大
- **平衡**：取决于脊柱区段

## 14.3 脊柱屈曲（前弯）

### 正常屈曲的动力学
```
初期：椎间盘和韧带伸展
    →椎间盘前部压力增加
    →神经元激活（Flexion-Relaxation）

中期：脊柱竖肌松弛，腹肌激活
    →更大的屈曲幅度

末期：韧带张力增加，提供弹性阻力
    →防止过度屈曲
```

### 屈曲的肌肉
- **原动肌**：腹直肌、斜方肌
- **协同肌**：腰大肌
- **对抗肌**：脊柱竖肌

## 14.4 脊柱伸展（后弯）

### 伸展的生物力学
- **椎间盘压力**：后部增加
- **关节突压力**：增加
- **韧带张力**：前纵韧带伸展

### 伸展的肌肉
- **原动肌**：脊柱竖肌、多裂肌
- **协同肌**：三角肌后部
- **对抗肌**：腹肌

### 伸展的限制因素
- **关节突撞击**：后位限制
- **韧带张力**：前纵韧带
- **椎间盘压力**：后部压力过大

## 14.5 脊柱侧屈（侧弯）

### 侧屈的特点
```
脊柱侧屈时，脊柱呈C形弯曲
    ↓
凸侧：椎体楔形、椎间盘减压
    ↓
凹侧：椎体收缩、椎间盘受压
```

### 侧屈的肌肉
| 肌肉 | 作用 | 特点 |
|------|------|------|
| 腰方肌 | 同侧侧屈 | 很强 |
| 腹斜肌 | 协助侧屈 | 协同作用 |
| 脊柱竖肌 | 协助侧屈 | 部分肌纤维参与 |

## 14.6 脊柱旋转（转身）

### 旋转的结构基础
- **椎间盘纤维环**：抵抗旋转应力
- **关节突**：引导旋转方向
- **旋转肌**：产生旋转力量

### 旋转的限制
- **纤维环**：能承受30°旋转
- **神经根**：旋转过度时可能压迫
- **椎间盘**：旋转时最容易受伤

### 旋转的肌肉
- **内旋**：腹内斜肌对侧纤维
- **外旋**：腹外斜肌同侧纤维
- **协调**：脊柱旋转肌、多裂肌

## 14.7 躯干肌肉的功能

### 核心稳定肌群

**腹肌系统**
- **腹直肌**：前屈肌，视觉上的"六块腹肌"
- **腹外斜肌**：侧屈和旋转
- **腹内斜肌**：侧屈和对侧旋转
- **腹横肌**：深层稳定肌，维持腹腔压力

**背部肌肉**
- **脊柱竖肌**：后伸肌，表浅
- **多裂肌**：深层稳定肌
- **竖棘肌**：长条肌肉，跨越多个脊椎

**侧面肌肉**
- **腰方肌**：侧屈和稳定
- **腰大肌**：髋屈和脊柱屈

### 肌肉的稳定性功能
- **维持脊柱对齐**：防止过度活动
- **分散载荷**：保护椎间盘
- **保护脊髓**：减少神经压迫风险

## 14.8 脊柱功能的联合运动

### 结合运动（复合运动）
脊柱在功能性活动中很少单独进行一个运动，通常是多个方向的组合：

**弯腰拿东西**
- 脊柱屈曲（主要）
- 髋部屈曲
- 轻微旋转（调整位置）

**转身看后方**
- 脊柱旋转（主要）
- 轻微侧屈
- 颈椎旋转补偿

## 要点总结

✓ **颈椎**旋转最灵活，**胸椎**旋转活动也很好，**腰椎**以屈伸为主
✓ **脊柱屈曲**时腹肌激活，**脊柱伸展**时脊柱竖肌激活
✓ **椎间盘**在脊柱运动中面临不同方向的应力
