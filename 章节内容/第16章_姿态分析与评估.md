# 第16章：姿态分析与评估

## 学习目标
- 理解正常姿态的标准
- 掌握姿态分析的方法
- 认识常见的姿态异常
- 理解姿态与功能障碍的关系

## 16.1 正常姿态的标准

### 侧面观
正常侧面姿态的"重力线"应该通过：
```
外耳道
        ↓
肩关节（肩峰）
        ↓
髋关节（大转子）
        ↓
膝关节（膝外侧）
        ↓
踝关节（踝外侧）
```

**特点**：身体各关节对齐，抗重力肌肉作用最小

### 前面观
- **头部**：正中位置，不侧倾
- **肩峰**：水平，左右对称
- **胸部**：对称，无扭转
- **骨盆**：水平，前上髂棘对齐
- **膝关节**：对齐，无内扣或外翻
- **足部**：脚尖向前，足弓正常

### 后面观
- **脊柱**：直线，无侧弯
- **肩胛骨**：贴近胸壁，对称
- **腰部**：对称，无侧凸
- **骨盆**：水平，后上髂棘对齐
- **下肢**：对齐，无X形或O形腿

## 16.2 姿态分析的方法

### 观察方法

**静态姿态观察**
1. 让患者保持自然站立
2. 从前、侧、后三个角度观察
3. 注意头颈、脊柱、骨盆、下肢的位置
4. 评估左右对称性

**动态姿态观察**
1. 观察走路姿势
2. 观察弯腰、转身等动作
3. 注意代偿模式
4. 识别功能限制

### 评估工具

**重力线参考**
- 使用垂线或参考线
- 建立一致的参考标准
- 便于拍照对比

**测量工具**
- 量角器：测量关节角度
- 测量尺：测量长度差异
- 照片对比：记录变化

**高级工具**
- 视频分析：逐帧观察
- 3D动作捕捉：精确测量
- 重心平台测试：静态平衡评估

## 16.3 常见的姿态异常

### 头部异常

**头前伸姿态**
- **特征**：头部向前，颈椎过伸展
- **原因**：长期低头、不良习惯
- **后果**：颈肩疼痛、头痛
- **改善**：加强颈部深层屈肌

**头侧倾**
- **特征**：头部一侧倾斜
- **原因**：肌肉紧张、神经根压迫
- **后果**：偏头痛、颈痛
- **改善**：拉伸紧张侧肌肉

### 脊柱异常

**圆肩姿态（上交叉综合征）**
- **特征**：胸椎后凸增加，肩向前内缩
- **原因**：胸肌紧张，背肌无力
- **后果**：肩痛、颈痛、呼吸困难
- **改善**：加强背肌，拉伸胸肌

**过度后凸（驼背）**
- **特征**：胸椎曲度过大，脊柱后弯
- **原因**：骨质疏松、姿态不良
- **后果**：呼吸受限、前方面积减少
- **改善**：加强背肌，改善姿态

**脊柱侧弯**
- **特征**：脊柱向一侧弯曲，呈S形或C形
- **原因**：先天性或习惯性
- **后果**：脊柱不对称、功能障碍
- **管理**：需医学评估，可能需支具

**过度前凸（腰椎前凸过多）**
- **特征**：腰椎曲度过大，髋向前倾
- **原因**：腰大肌紧张，腹肌无力
- **后果**：腰痛、下肢压力增加
- **改善**：加强腹肌，拉伸髋屈肌

### 骨盆异常

**骨盆前倾**
- **特征**：骨盆向前旋转，腰椎前凸
- **原因**：腰大肌紧张，腹肌和臀肌无力
- **后果**：腰痛、膝痛
- **改善**：加强腹肌和臀肌

**骨盆后倾**
- **特征**：骨盆向后旋转，腰椎平坦
- **原因**：腹肌过紧，腰大肌拉长被弱化
- **后果**：腰椎压力分布异常
- **改善**：拉伸腹肌，加强腰大肌

**骨盆倾斜**
- **特征**：左右骨盆高度不同
- **原因**：下肢长度差异、肌肉失衡
- **后果**：脊柱侧弯、下肢代偿
- **改善**：根据原因进行针对性治疗

### 下肢异常

**膝过度伸展（超伸）**
- **特征**：站立时膝向后超伸
- **原因**：股四头肌过度紧张或腘绳肌无力
- **后果**：膝关节应力增加
- **改善**：松弛股四头肌，加强腘绳肌

**膝内扣（内翻）**
- **特征**：膝向内倾斜
- **原因**：髋外旋无力，内旋肌紧张
- **后果**：膝关节异常应力
- **改善**：加强髋外旋肌，拉伸内旋肌

**足外翻**
- **特征**：脚向外转，足弓塌陷
- **原因**：足部肌肉无力，脚踝不稳
- **后果**：足部疼痛，步行困难
- **改善**：加强足部肌肉，增强踝部稳定

**足内翻**
- **特征**：脚向内转，高弓足
- **原因**：足部肌肉紧张，灵活性不足
- **后果**：脚踝易扭伤，外侧足痛
- **改善**：拉伸足部肌肉，增强灵活性

## 16.4 姿态与功能障碍的关系

### 姿态异常导致的功能障碍链

**例1：头前伸** 
```
头前伸 → 颈椎后伸增加 → 颈椎间盘压力增加 
      → 神经根压迫 → 肩臂疼痛、麻木
```

**例2：骨盆前倾**
```
骨盆前倾 → 腰椎前凸增加 → 椎间盘后方压力增加
        → 后方韧带张力增加 → 腰痛加重
```

**例3：膝内扣**
```
膝内扣 → 膝关节轴线异常 → 髌骨跟踪异常
      → 髌股关节压力不均 → 膝前痛
```

## 16.5 姿态评估的临床应用

### 诊断线索
- 识别可能的问题区域
- 指导进一步的检查
- 帮助诊断

### 治疗计划制定
- 确定重点改善区域
- 制定针对性治疗
- 评估预后

### 预防性干预
- 识别早期异常
- 进行预防性训练
- 改善健康状况

## 16.6 姿态改善的原则

### 评估优先
1. 确定主要问题
2. 识别代偿模式
3. 了解原因

### 分层治疗
1. 第一步：增加灵活性（拉伸紧张肌肉）
2. 第二步：加强肌肉（针对无力肌肉）
3. 第三步：建立新习惯（改变姿态模式）

### 循序渐进
- 不要一次改变太多
- 允许身体适应
- 逐步加强难度

## 要点总结

✓ 正常姿态遵循**重力线原则**，各关节应对齐
✓ 姿态异常通常由**肌肉失衡**引起
✓ 姿态改善需要**灵活性、力量和习惯**的综合改善
