# 第7章：腕部与手部解剖学

## 学习目标
- 掌握腕部的复杂骨骼结构
- 理解腕部的多轴运动特性
- 认识手部精细控制的机制
- 理解腕手协调功能

## 7.1 腕部骨骼结构

### 腕骨的排列
腕部由8块小骨头组成，分两排：

**近端排**（从拇侧到尺侧）
- 舟状骨、月骨、三角骨、豌豆骨

**远端排**（从拇侧到尺侧）
- 大多角骨、小多角骨、头状骨、钩状骨

### 腕骨的功能特点
- **数量多**：8块骨头提供灵活性
- **排列复杂**：便于多方向运动
- **易扭伤**：韧带容易受伤

### 手部骨骼
- **掌骨**：5块，形成掌弓
- **指骨**：14块，形成手指骨架
- **结构**：近节、中节、远节

## 7.2 腕部关节与运动学定义

腕关节（桡腕关节）是一个双轴的椭圆关节，其运动通常发生在两个相互垂直的平面内。

### 7.2.1 腕部的运动维度

| 运动类型 | 发生平面 | 转动轴 | 正常范围 | 协调肌肉 |
| :--- | :--- | :--- | :--- | :--- |
| **屈曲/伸展** | **矢状面** | **冠状轴** | 屈70-80° / 伸60-70° | 桡/尺侧腕屈肌、掌长肌 / 桡/尺侧腕伸肌 |
| **桡偏/尺偏** | **冠状面** | **矢状轴** | 桡偏15-20° / 尺偏30-40° | 桡侧腕屈/伸肌 / 尺侧腕屈/伸肌 |
| **环转** | **综合面** | **多轴** | 上述运动的结合 | 屈、伸、偏肌群的顺序激活 |

### 7.2.2 功能位与稳定性
- **解剖学中立位**：在标准解剖学姿势下，手掌朝前，腕部处于各轴线的交点。
- **功能位 (Functional Position)**：腕部背伸20-30°，尺偏约10°。此位置下，指屈肌的长度-张力关系达到最优，握力最强。
- **腕间关节的滑动**：除了整体的摆动运动，腕骨之间还存在微小的滑动，这增加了关节的整体灵活性和减震能力。

## 7.3 腕部肌肉

### 腕屈肌
| 肌肉 | 起点 | 止点 | 功能 |
|------|------|------|------|
| 桡侧腕屈肌 | 肱骨内上髁 | 2、3掌骨 | 屈腕、桡偏 |
| 尺侧腕屈肌 | 肱骨内上髁 | 豌豆骨、尺骨 | 屈腕、尺偏 |
| 掌长肌 | 肱骨内上髁 | 掌腱膜 | 屈腕、握拳 |

### 腕伸肌
| 肌肉 | 起点 | 止点 | 功能 |
|------|------|------|------|
| 桡侧腕伸肌长头 | 肱骨外上髁 | 2掌骨 | 伸腕、桡偏 |
| 桡侧腕伸肌短头 | 外侧韧带 | 3掌骨 | 伸腕 |
| 尺侧腕伸肌 | 肱骨外上髁 | 5掌骨 | 伸腕、尺偏 |

### 前臂旋转肌
- **旋后圆肌**：腕下方旋后
- **旋前方肌**：腕下方旋前

## 7.4 手部肌肉与控制

### 内在肌（Intrinsic Muscles）
- **位置**：手掌内部
- **功能**：精细控制，如弹琴、书写
- **类型**：骨间肌、蚓状肌、鱼际肌、小鱼际肌

### 外在肌（Extrinsic Muscles）
- **位置**：前臂，肌腱延伸至手
- **功能**：产生主要力量，如握拳
- **类型**：指屈肌、指伸肌

### 握拳的协调
```
握拳 = 指屈肌（力量） + 内在肌（精细控制）
```

## 7.5 手部的功能组织

### 手的分区
- **拇指侧**（鱼际）：力量和灵活性
- **中部**：稳定和控制
- **尺侧**：力量支持

### 重要的功能抓握方式

| 抓握类型 | 特点 | 用途 |
|---------|------|------|
| 力握 | 全掌参与 | 拿重物 |
| 精细握 | 拇指对指 | 拿细物 |
| 钩握 | 四指弯曲 | 提东西 |
| 侧握 | 拇指与食指侧面 | 推东西 |

## 7.6 腕手的协调功能

### 腕部位置对握力的影响
```
握力最强 ← 腕部中立位置
握力较弱 ← 腕部过屈或过伸
```

### 运动学链
- **肩部位置**影响肘部效率
- **肘部位置**影响腕部活动
- **腕部位置**影响握力
- **手指位置**决定精度

## 7.7 常见腕手功能障碍

### 腕部扭伤
- **原因**：跌倒时腕部过度背伸
- **常见伤**：舟状骨骨折
- **康复**：稳定性和活动度恢复

### 鼠标手（腕管综合征）
- **原因**：正中神经受压
- **症状**：拇指、食指、中指麻木
- **治疗**：姿态调整、神经松动

### 手部无力
- **原因**：颈椎病、周围神经病变
- **评估**：握力测试、精细动作测试

## 要点总结

✓ 腕部由**8块腕骨**组成，具有**多轴运动**能力
✓ 手部**外在肌**提供力量，**内在肌**提供精度
✓ **腕部位置**对**握力**有重要影响
