# 第3章：肌肉系统解剖学基础

## 学习目标
- 掌握骨骼肌的组织结构
- 理解肌肉收缩的机制
- 认识肌肉类型及其特点
- 理解肌肉的功能分类

## 3.1 骨骼肌的组织结构

### 宏观结构
```
肌肉纤维束 ← 肌束膜（Perimysium）
  ↓
肌纤维 ← 肌内膜（Endomysium）
  ↓
肌动蛋白与肌球蛋白
```

### 微观结构
- **肌原纤维**：收缩单位
- **肌节**（Sarcomere）：最小收缩单位
  - 由Z盘界定
  - 包含厚丝（肌球蛋白）和细丝（肌动蛋白）

### 肌肉-腱单位
| 结构 | 功能 |
|------|------|
| 肌腱 | 连接肌肉与骨骼 |
| 肌腱鞘 | 减少摩擦 |
| 腱膜 | 增大附着面积 |

## 3.2 肌肉收缩的机制

### 滑动学说
1. **激活**：神经信号→肌动蛋白激活
2. **交叉桥形成**：肌球蛋白头连接肌动蛋白
3. **力冲程**：肌球蛋白头牵拉肌动蛋白
4. **解离**：ATP提供能量分离交叉桥
5. **复位**：肌球蛋白头复位，准备下一个循环

### 能量代谢
- **ATP-PC系统**：最快（0-10秒）
- **无氧糖酵解**：中等（10-60秒）
- **有氧代谢**：可持续（>60秒）

## 3.3 肌肉纤维类型

### 快缩纤维（Type II）
| 特征 | 详情 |
|------|------|
| 收缩速度 | 快 |
| 力量 | 强 |
| 耐力 | 低 |
| 代谢 | 无氧为主 |
| 用途 | 爆发性运动 |

### 慢缩纤维（Type I）
| 特征 | 详情 |
|------|------|
| 收缩速度 | 慢 |
| 力量 | 弱 |
| 耐力 | 高 |
| 代谢 | 有氧为主 |
| 用途 | 耐力运动、姿态维持 |

## 3.4 肌肉的功能分类

### 收缩类型

**向心收缩**（Concentric）
- 肌肉缩短
- 产生正功
- 例：上臂弯曲时肱二头肌

**离心收缩**（Eccentric）
- 肌肉拉长
- 产生制动
- 例：下楼时股四头肌

**等长收缩**（Isometric）
- 肌肉长度不变
- 维持姿态
- 例：挺直脊背时腹肌

### 运动学功能

**原动肌**（Agonist）
- 主要产生运动
- 如：上臂屈曲时的肱二头肌

**对抗肌**（Antagonist）
- 产生相反运动
- 如：上臂屈曲时的肱三头肌

**协同肌**（Synergist）
- 辅助原动肌
- 消除不必要运动

**稳定肌**（Stabilizer）
- 固定某些关节
- 提供稳定基础

## 3.5 主要肌肉群的功能

### 核心肌群（Core Muscles）
- **腹直肌**：脊柱屈曲
- **腹横肌**：腹腔压力维持
- **竖脊肌**：脊柱伸展
- **多裂肌**：脊柱稳定

### 上肢主要肌肉
- **肩部**：冈上肌、冈下肌、三角肌
- **肘部**：肱二头肌、肱三头肌
- **腕部**：腕屈肌、腕伸肌

### 下肢主要肌肉
- **髋部**：臀大肌、臀中肌、腰大肌
- **膝部**：股四头肌、腘绳肌
- **踝部**：小腿三头肌、胫骨前肌

## 3.6 肌肉长度与张力的关系

### 长度-张力曲线
```
最大张力
    |     ╱╲
张力|    ╱  ╲
    |   ╱    ╲
    |__╱______╲___
      最优长度   肌肉长度
```

### 临床意义
- 过度拉长→张力下降
- 过度缩短→张力下降
- 最优长度→最大张力

## 要点总结

✓ 肌肉由**肌纤维**组成，最小收缩单位是**肌节**
✓ **快缩纤维**用于爆发，**慢缩纤维**用于耐力
✓ **向心、离心、等长**三种收缩方式各有用途
