# 第13章：脊柱解剖学基础

## 学习目标
- 掌握脊柱的分段解剖结构
- 理解椎体和椎间盘的结构
- 认识脊柱的稳定性机制
- 理解脊柱的神经支配

## 13.1 脊柱的分段结构

### 脊柱的组成
脊柱共由33块椎骨组成，分为5个区段：

| 区段 | 椎骨数 | 特点 |
|------|--------|------|
| 颈椎 | 7块（C1-C7） | 最灵活，易受伤 |
| 胸椎 | 12块（T1-T12） | 中等灵活，固定性好 |
| 腰椎 | 5块（L1-L5） | 承重最大，活动受限 |
| 骶椎 | 5块（融合） | 完全融合，无活动 |
| 尾椎 | 4块（退化） | 退化残留，无功能 |

### 脊柱的正常曲线
```
        颈曲（前凸）← 生理性
        ↓
        胸曲（后凸）← 生理性
        ↓
        腰曲（前凸）← 生理性
        ↓
        骶曲（后凸）← 固定
```

**功能**：吸收冲击、分散压力、增加灵活性

## 13.2 椎骨的结构

### 椎骨的基本部分
```
椎体（前方，承重）
    ↓
椎弓根（连接部）
    ↓
椎弓（后方，保护）
    ↓
    ├─棘突（后方突出）
    ├─横突（两侧突出）
    └─关节突（形成关节）
```

### 椎体的特点
- **功能**：主要承重结构
- **排列**：上下紧密相连
- **尺寸**：从颈到腰逐渐增大

### 椎弓的功能
- **保护**：保护脊髓和神经根
- **支持**：参与关节形成
- **肌肉附着**：强大肌肉的起止点

## 13.3 椎间盘的结构与功能

### 椎间盘的组成
```
纤维环（外层）← 纤维软骨，抵抗剪切力
    ↓
髓核（中心）← 胶状物质，吸收压力
```

### 椎间盘的力学特性
- **承压**：髓核吸收垂直压力
- **抵抗剪切**：纤维环增加剪切阻力
- **弹性**：椎间盘充当减震器

### 椎间盘的变化
- **日间**：压力作用下，水分被挤出，身高降低1-2cm
- **夜间**：休息时，水分被吸收，身高恢复

## 13.4 脊柱的关节结构

### 椎体间关节（椎间盘）
- **类型**：纤维软骨关节
- **活动性**：很小，但累积产生大幅度
- **特点**：是脊柱运动的主要贡献者

### 关节突关节（后关节）
- **位置**：椎弓后方
- **类型**：平面关节
- **功能**：
  - 引导脊柱运动方向
  - 防止过度活动
  - 分担部分载荷

### 13.4.3 脊柱的活动维度

脊柱作为典型的多轴复合体，在三个相互垂直的平面内均有活动：
- **矢状面**：屈曲（前弯）与伸展（后弯），围绕**冠状轴**运动。
- **冠状面**：左右侧屈（侧弯），围绕**矢状轴**运动。
- **横断面**：左右旋转（转身），围绕**垂直轴**运动。

## 13.5 脊柱的稳定性结构

### 被动稳定系统
- **椎间盘**：抵抗剪切和弯曲
- **关节突**：引导运动和防止脱位
- **韧带**：连接椎骨，限制过度运动

### 主要韧带

| 韧带 | 位置 | 功能 |
|------|------|------|
| 前纵韧带 | 椎体前方 | 防止过度伸展 |
| 后纵韧带 | 椎体后方 | 防止脊髓压迫 |
| 黄韧带 | 椎弓间 | 防止过度屈曲 |
| 棘间韧带 | 棘突间 | 支持脊柱 |
| 横突间韧带 | 横突间 | 侧向稳定 |

### 主动稳定系统
- **脊柱竖肌**：后伸肌肉
- **腹肌**：前屈和稳定
- **斜方肌**：上背部稳定
- **腰方肌**：侧屈和稳定

## 13.6 脊柱的神经支配

### 脊髓的分段
- **颈脊髓**（C1-C8）：8对脊神经
- **胸脊髓**（T1-T12）：12对脊神经
- **腰脊髓**（L1-L5）：5对脊神经
- **骶脊髓**（S1-S5）：5对脊神经

### 神经根的通道
```
从脊髓出发 → 通过椎间孔 → 分布到躯干和四肢
```

### 脊神经的压迫
- **椎间盘突出**：压迫下位神经根
- **骨刺增生**：刺激神经根
- **椎体滑脱**：改变椎间孔形状

## 13.7 每个脊柱区段的特点

### 颈椎的特点
- **活动度**：最大
- **承重**：最小
- **易损伤**：颈椎病常见
- **神经支配**：上肢神经来源

### 胸椎的特点
- **相对固定**：受肋骨固定
- **活动度**：受限
- **旋转**：旋转幅度最大
- **支撑**：支持胸腔脏器

### 腰椎的特点
- **承重**：最大
- **活动度**：屈伸活动大
- **易问题**：腰痛最常见
- **厚度**：椎间盘最厚

### 骶椎的特点
- **融合**：5块椎骨融合成一块骨头
- **功能**：连接脊柱和骨盆
- **无活动**：完全固定

## 要点总结

✓ 脊柱由**33块椎骨**分**5个区段**组成
✓ **椎间盘和关节突**是脊柱运动的主要结构
✓ **主动和被动**系统共同维持脊柱稳定
