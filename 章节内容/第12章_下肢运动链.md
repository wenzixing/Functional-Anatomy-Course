# 第12章：下肢运动链

## 学习目标
- 理解下肢作为整体运动系统
- 掌握下肢各关节的协调原理
- 认识步行的运动学基础
- 理解下肢运动的常见代偿模式

## 12.1 下肢运动链的概念

### 开放链与闭合链
- **开放链**：末端自由（如抬腿）
- **闭合链**：末端固定（如站立、步行）
- **下肢运动**：主要为闭合链（承重）

### 下肢链的结构
```
髋关节 → 膝关节 → 踝关节 → 足部
  |        |        |       |
承重    传力    适应   推进
```

## 12.2 下肢运动链的稳定性

### 静态稳定性（站立）

**单腿站立**
- **髋外展肌**：维持骨盆水平
- **核心肌群**：控制脊柱稳定
- **踝部肌肉**：微调平衡

**双腿站立**
- **对称分布**：体重均匀分配
- **自动调节**：无需意识控制
- **本体感受反馈**：持续微调

### 动态稳定性（运动）
- **临界稳定期**：单脚支撑时最不稳定
- **肌肉协调**：完美的肌肉序列
- **代偿机制**：自动调整

## 12.3 步行的生物力学

### 步行周期的定义
```
一个完整步行周期 = 从一只脚触地 → 同一脚再次触地
                  ≈ 1秒（正常步速）
```

### 步行周期的两个主要阶段

**支撑期（约60%）**
- 脚与地面接触
- 承重和推进
- 关键肌肉：臀肌、股四头肌、小腿三头肌

**摆动期（约40%）**
- 脚离开地面
- 下肢前摆准备
- 关键肌肉：腰大肌、股直肌

### 每个阶段的细分

**触地缓冲（0-10%）**
- 脚跟先着地
- 膝关节小幅屈曲
- 踝部背伸控制

**加载响应（0-10%）**
- 足部完全接地
- 膝屈曲加深
- 髋关节伸展加速

**中支撑（10-30%）**
- 身体向前移动
- 对侧腿开始摆动
- 单腿支撑稳定性最重要

**摆动准备（30-50%）**
- 脚趾离地准备
- 髋关节伸展加速
- 小腿三头肌推动

**初摆动（50-73%）**
- 腿快速向前摆
- 膝关节屈曲
- 髋关节屈曲加速

**中摆动（73-87%）**
- 腿继续前摆
- 膝关节逐步伸展
- 为触地做准备

**末摆动（87-100%）**
- 腿完全伸展
- 准备触地
- 膝关节锁定

## 12.4 下肢关节的多平面协调

在步行过程中，下肢三个主要关节并不是独立工作的，而是在三个解剖面内进行高度复杂的耦合运动。

### 12.4.1 矢状面内的协同（主运动平面）
这是步行中最显著的平面，负责推动身体向前：
- **触地 (Initial Contact)**：膝关节锁定（接近0°），踝背伸处于中立位。
- **加载响应 (Loading Response)**：为了缓冲冲击，膝关节发生15-20°的被动屈曲，伴随踝关节的跖屈（控制性下放）。
- **推进 (Propulsion)**：髋、膝、踝三者同步向后推（伸展），将身体力量转化为向前的动量。

### 12.4.2 冠状面与横断面的稳定
虽然步行的主要方向是向前的，但平稳性取决于另外两个平面的控制：
- **骨盆控制 (冠状面)**：在单腿支撑期，由于身体重心外偏移，支撑侧的臀中肌必须强力收缩（外展功能），以防止骨盆向对侧下沉（围绕**矢状轴**）。
- **旋转微调 (横断面)**：骨盆和股骨在水平面内存在交替的旋内与旋外运动，以优化力的传递路径。

### 12.4.3 下肢链条中的力量传递顺序
1. **吸收段**：足与踝（通过内翻/外翻微调）适应地面。
2. **缓冲段**：膝关节通过微屈吸收重力势能。
3. **驱动段**：髋部肌肉作为“发动机”产生主要推力。

### 下肢链条中的力量传递
1. **起点**：臀肌和核心肌群
2. **中点**：膝部力量过渡
3. **终点**：足部推进

## 12.5 下肢的步行变量

### 步长与步频
- **步长**：两次同侧脚触地的距离
- **步频**：单位时间内的步数
- **关系**：步速 = 步长 × 步频

### 步行速度的影响
- **慢步**：膝屈曲减少，步长缩短
- **快步**：膝屈曲增加，步长增长
- **跑步**：出现腾空期，关节活动增大

### 不同年龄的步行特点
- **儿童**：步长短，步频快
- **成人**：步长标准，步频中等
- **老人**：步长短，步频慢，支撑期增加

## 12.6 常见下肢代偿模式

### Trendelenburg步态
**特征**：单脚支撑时对侧骨盆下沉
**原因**：支撑侧臀中肌无力
**后果**：膝盖和踝部代偿，增加压力

### 膝过度伸展步态
**特征**：支撑期膝超伸
**原因**：股四头肌过度收缩或腿部肌肉无力
**后果**：膝关节应力增加

### 内翻步态
**特征**：脚向内转，膝内扣
**原因**：髋内旋肌肉紧张，外旋肌肉无力
**后果**：膝关节异常应力，易导致膝痛

### 脚内翻步态
**特征**：脚向内转
**原因**：足部肌肉失衡或脚部对齐问题
**后果**：踝部不稳定，足弓塌陷

### 步态分析
- **观察步行**：识别明显异常
- **视频分析**：详细评估模式

## 12.8 下肢功能优化

### 力量训练
- **开链训练**：如腿部推举
- **闭链训练**：如深蹲、跨步
- **功能性训练**：如楼梯、不稳定面

### 活动度训练
- **膝部柔韧**：腘绳肌、股四头肌拉伸
- **踝部灵活**：踝部各方向活动
- **髋部灵活**：髋关节全方位运动

### 本体感受训练
- **平衡训练**：不稳定表面站立
- **单腿训练**：复杂运动单腿完成
- **神经肌肉训练**：增强控制

## 要点总结

✓ 下肢是**承重、推进和运动**的综合系统
✓ **步行**是最复杂的日常功能活动
✓ 识别和纠正**代偿模式**对下肢健康至关重要
