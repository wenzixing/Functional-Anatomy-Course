# 第2章：骨骼系统解剖学基础

## 学习目标
- 掌握骨骼的基本结构与功能
- 理解骨骼的分类与特点
- 熟悉全身主要骨骼标志
- 理解骨骼对运动的支持作用

## 2.1 骨骼的基本结构

### 宏观结构
- **骨干**（Diaphysis）：中间部分，承受主要应力
- **骨骺**（Epiphysis）：两端膨大部分，形成关节面
- **骨转子**（Trochanter）：大的隆起，肌肉附着点
- **骨嵴**（Crest）：骨表面的隆起线

### 微观结构
- **密质骨**：表面致密，强度高
- **松质骨**：内部疏松，轻质
- **骨髓**：中央腔隙，造血功能

## 2.2 骨骼的功能

### 力学功能
| 功能 | 描述 | 临床意义 |
|------|------|---------|
| 支持 | 支撑体重和肌肉 | 骨密度降低导致易折 |
| 杠杆 | 与肌肉协作产生运动 | 杠杆臂长度影响功效 |
| 保护 | 保护内脏器官 | 骨折风险评估 |

### 生理功能
- 造血功能（骨髓）
- 矿物质储存（钙、磷）

## 2.3 骨骼的分类

### 按形态分类

**长骨**（Long bones）
- 结构：骨干 + 骨骺 + 骨转子
- 例如：股骨、肱骨、胫骨
- 功能：承受纵向应力

**短骨**（Short bones）
- 形状：立方体
- 例如：腕骨、跗骨
- 功能：灵活支撑和活动

**扁骨**（Flat bones）
- 形状：扁平
- 例如：肋骨、肩胛骨、骨盆
- 功能：保护和附着肌肉

**不规则骨**（Irregular bones）
- 形状：复杂
- 例如：椎骨、髋骨
- 功能：支持和保护

**籽骨**（Sesamoid bones）
- 特点：位于肌腱内
- 例如：髌骨
- 功能：改变力的方向

## 2.4 全身主要骨骼标志

### 脊柱
- **颈椎**（C1-C7）：最灵活，易受伤
- **胸椎**（T1-T12）：相对固定
- **腰椎**（L1-L5）：承受压力最大
- **骶椎**：融合体
- **尾椎**：退化残留

### 上肢骨骼
- **肩关节**：肱骨头与肩胛骨盂
- **肘关节**：滑车与尺骨半月形切迹
- **腕部**：8块腕骨排列
- **手部**：掌骨与指骨

### 下肢骨骼
- **髋关节**：股骨头与髋臼
- **膝关节**：股骨髁与胫骨平台
- **踝部**：胫腓与距骨
- **足部**：7块跗骨 + 跖骨 + 趾骨

## 2.5 骨骼在运动中的作用

### 杠杆系统
- **第一类杠杆**：支点在力和阻力之间（如颈后伸）
- **第二类杠杆**：阻力在支点与力之间（如踮脚）
- **第三类杠杆**：力在支点与阻力之间（多数骨骼肌运动）

### 支持功能
- 骨质疏松影响支持能力
- 应力集中点易发生骨折
- 关节面磨损导致功能障碍

## 2.6 骨-肌肉相互作用

### 肌肉附着点
- **起点**（Origin）：通常固定，靠近中线
- **止点**（Insertion）：运动，远离中线
- **杠杆臂**：决定肌肉效率

### 生物力学优势
$$\text{力的优势} = \frac{\text{肌肉到关节轴的距离}}{\text{阻力到关节轴的距离}}$$

## 要点总结

✓ 骨骼由**密质骨和松质骨**组成
✓ 骨骼的基本功能是**支持、杠杆和保护**
✓ 五种主要骨骼分类体现了**形态与功能的统一**
