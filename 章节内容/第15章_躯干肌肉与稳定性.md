# 第15章：躯干肌肉与稳定性

## 学习目标
- 理解躯干肌肉的分层结构
- 掌握核心稳定性的概念
- 认识躯干肌肉的功能分类
- 理解脊柱稳定性的生理学基础

## 15.1 躯干肌肉的解剖分层

### 表浅层（功能层）

**腹直肌**
- **位置**：腹前正中
- **起点**：耻骨
- **止点**：胸骨、肋软骨
- **功能**：脊柱屈曲、核心压力
- **特点**：最容易看到的腹肌

**脊柱竖肌**
- **位置**：脊柱两侧
- **起点**：骨盆、腰椎
- **止点**：胸椎、颈椎
- **功能**：脊柱伸展、侧屈
- **特点**：长条肌肉，跨越多个椎骨

**腹外斜肌**
- **位置**：腹部两侧外层
- **起点**：肋骨下方
- **止点**：骨盆、腹部中线
- **功能**：脊柱屈曲、对侧旋转、侧屈
- **特点**：纤维呈对角线方向

### 中层（过渡层）

**腹内斜肌**
- **位置**：腹部两侧内层
- **功能**：同侧旋转、脊柱屈曲、侧屈
- **特点**：纤维方向与外斜肌相反（协调运动）

**腰方肌**
- **位置**：脊柱侧方深层
- **起点**：骨盆、脊柱下部
- **止点**：肋骨下方
- **功能**：侧屈、脊柱稳定
- **特点**：非常强的肌肉

### 深层（稳定层）

**腹横肌**
- **位置**：腹部最深层
- **走向**：横向排列
- **功能**：维持腹腔压力、脊柱稳定
- **特点**：深层稳定肌，最重要

**多裂肌**
- **位置**：脊柱后侧深层
- **走向**：斜向上方
- **功能**：脊柱伸展、局部稳定
- **特点**：在每个脊椎都有

**竖棘肌**
- **位置**：脊柱深层
- **功能**：脊柱伸展、稳定
- **特点**：长的、中的、短的三组

## 15.2 核心稳定性的概念

### 核心的定义
**核心**是指脊柱周围的肌肉系统，包括：
- 腹肌（所有层）
- 背部肌肉（脊柱竖肌、多裂肌）
- 盆底肌群
- 膈肌
- 腰方肌

### 核心稳定性的功能
1. **保护脊柱**：防止过度运动
2. **分散载荷**：保护椎间盘
3. **传递力量**：连接上下肢
4. **维持姿态**：抗重力
5. **增强表现**：提升运动效率

## 15.3 脊柱稳定性的生理学机制

### 被动稳定系统
- **椎骨**：骨架结构
- **椎间盘**：减震垫
- **韧带**：被动约束
- **作用范围**：约30%的稳定性

### 主动稳定系统
- **肌肉**：主动收缩
- **肌腱**：力量传递
- **神经控制**：精确调节
- **作用范围**：约70%的稳定性

### 动态稳定性
```
脊柱稳定 = 骨骼结构 + 韧带张力 + 肌肉协调
        = 被动系统 + 主动系统
```

## 15.4 躯干肌肉的功能分类

### 按功能分层

**稳定肌（Stability Muscles）**
- **特点**：深层、单关节、慢缩纤维
- **功能**：维持脊柱对齐
- **代表**：腹横肌、多裂肌
- **训练方法**：低负荷、高耐力

**运动肌（Mobility Muscles）**
- **特点**：表浅、多关节、快缩纤维
- **功能**：产生运动
- **代表**：腹直肌、脊柱竖肌
- **训练方法**：高负荷、动态运动

### 稳定肌与运动肌的失衡
```
稳定肌无力 + 运动肌紧张 = 腰椎不稳定
        ↓
    容易发生损伤
        ↓
    疼痛和功能障碍
```

## 15.5 躯干肌肉的协调模式

### 脊柱屈曲时的肌肉激活

**动作初期**
- 腹直肌启动
- 脊柱竖肌逐步放松（Flexion-Relaxation现象）
- 腹横肌预激活（维持腹腔压力）

**动作中期**
- 腹斜肌参与
- 腹直肌增加收缩强度
- 脊柱竖肌完全松弛

**动作末期**
- 韧带张力增加
- 脊柱竖肌最小激活（防止过度屈曲）

### 脊柱伸展时的肌肉激活

**初期启动**
- 脊柱竖肌激活
- 腹肌预张力维持

**主动伸展**
- 脊柱竖肌集中收缩
- 多裂肌增加稳定性
- 腹肌协同稳定

**末期停止**
- 关节突接触增加
- 韧带张力增加
- 肌肉逐步放松

## 15.6 常见的躯干肌肉失衡

### 上交叉综合征
**特点**：
- 胸肌紧张，菱形肌无力
- 上背部圆弯，颈前伸
- 原因：长期久坐，不良姿态

### 下交叉综合征
**特点**：
- 腰大肌和腰方肌紧张
- 腹肌无力，臀肌无力
- 原因：长期久坐，缺乏运动
- 后果：腰痛、腰椎不稳定

### 核心肌群无力综合征
**特点**：
- 深层稳定肌无力
- 腹横肌激活延迟
- 原因：姿态不良、缺乏训练
- 后果：脊柱不稳定、疼痛

## 15.7 核心稳定性训练

### 基础阶段

**腹横肌激活**
- 仰卧，膝屈，吸气准备
- 吐气时收缩腹横肌（肚脐向脊柱靠近）
- 维持5-10秒，重复10次

**多裂肌激活**
- 四点支撑位置
- 对侧上肢和下肢抬起
- 维持稳定5秒，交替进行

### 进阶阶段

**动态稳定性训练**
- 平板支撑
- 鸟狗式变式
- 瑞士球训练

### 高阶阶段

**功能性核心训练**
- 不稳定面训练
- 旋转运动训练
- 复合动作训练

## 15.8 躯干稳定性的临床应用

### 在康复中的应用
- **腰痛管理**：强化深层稳定肌
- **术后恢复**：逐步重建核心
- **预防再伤**：建立稳定基础

### 在运动中的应用
- **提升表现**：增强力量传递
- **预防损伤**：提高脊柱保护
- **改善技术**：改善运动效率

## 要点总结

✓ 躯干肌肉分为**稳定肌和运动肌**，各司其职
✓ **深层稳定肌**对脊柱保护最重要
✓ **核心稳定性**是所有运动的基础
