# 第11章：踝足部解剖学

## 学习目标
- 掌握踝足部的复杂骨骼结构
- 理解踝部的运动特点
- 认识足部的功能结构
- 理解踝足功能与步行的关系

## 11.1 踝足部的骨骼结构

### 跗骨的组成
踝足部由7块跗骨组成：

| 骨骼 | 位置 | 功能 |
|------|------|------|
| 距骨 | 上方，连接胫腓 | 承受体重 |
| 跟骨 | 后下方 | 支撑和杠杆 |
| 舟状骨 | 内侧中部 | 足弓支持 |
| 立方骨 | 外侧中部 | 足弓支持 |
| 三个楔形骨 | 前内侧 | 足弓结构 |

### 跖骨与趾骨
- **跖骨**：5块，形成足弓
- **趾骨**：14块，形成足指结构

## 11.2 踝关节的结构与功能

### 踝关节（胫距关节）
- **关节面**：胫腓骨与距骨
- **类型**：滑车关节（单轴）
- **运动**：背伸与跖屈

### 背伸与跖屈的范围
- **背伸**：10-15°
- **跖屈**：45-50°
- **特点**：跖屈范围大，但力量在背伸时有不利

### 亚距下关节（距下关节）
- **功能**：内翻/外翻
- **范围**：内翻30°，外翻15°
- **作用**：适应不平坦地面

## 11.3 足部的结构与功能

### 足弓的构成

**内侧纵弓（主要）**
```
跟骨 → 舟状骨 → 楔形骨 → 跖骨
```
- **支撑**：跟骨和第1-3跖骨头
- **弹性**：吸收冲击
- **高度**：足内侧隆起

**外侧纵弓（次要）**
```
跟骨 → 立方骨 → 跖骨
```
- **支撑**：跟骨和第4-5跖骨头
- **特点**：较低，接触地面

**横弓**
- **构成**：多块跗骨和跖骨横向排列
- **作用**：增加足部稳定性

### 足弓的支持结构
- **骨骼**：楔形结构
- **韧带**：足底韧带、跟立方韧带
- **肌肉**：足底肌肉维持弓形

## 11.4 踝足部主要肌肉

### 背伸肌
| 肌肉 | 位置 | 功能 | 特点 |
|------|------|------|------|
| 胫骨前肌 | 前外侧 | 背伸、内翻 | 最强 |
| 趾长伸肌 | 前外侧 | 背伸、伸足趾 | 辅助 |
| 腓骨长肌 | 外侧 | 跖屈、外翻 | 重要稳定肌 |

### 跖屈肌
| 肌肉 | 位置 | 功能 |
|------|------|------|
| 小腿三头肌 | 后侧 | 跖屈（最强） |
| 腓肠肌 | 表浅 | 跖屈、膝屈 |
| 比目鱼肌 | 深层 | 跖屈 |
| 胫骨后肌 | 深层 | 跖屈、内翻 |

### 内翻肌
- **胫骨后肌**：最强
- **腓肠肌**：辅助
- **趾长屈肌**：辅助

### 外翻肌
- **腓骨长肌**：主要
- **腓骨短肌**：辅助

## 11.5 足部的功能特性

### 足部的双重功能
```
支撑与稳定性 ←→ 灵活性与适应性
     （硬）         （柔）
```

**触地时**：足部充分负重，形成稳定基础
**推进时**：足部灵活发力，推动身体前进

### 足部在步行中的作用

**触地阶段**
- 脚跟先着地
- 足部内翻，吸收冲击
- 足弓下陷

**加载响应**
- 足部逐步接受体重
- 足弓和踝部稳定
- 小腿肌肉离心收缩

**中摆动期**
- 足部完全负重
- 足弓形成坚固基础
- 准备推进

**推进期**
- 脚趾离地准备
- 小腿三头肌集中收缩
- 足部提供推进力

## 11.6 常见踝足部功能障碍

### 踝关节扭伤
- **原因**：内翻损伤（最常见）
- **症状**：外侧疼痛、肿胀、活动困难
- **康复**：早期保护，逐步进行本体感受训练

### 足弓塌陷（平足）
- **原因**：足弓支持肌肉无力
- **症状**：足内侧疼痛，步行困难
- **治疗**：足底肌肉强化，足弓支持

### 足弓过高（高弓足）
- **原因**：足底肌肉过度紧张
- **症状**：外侧脚疼痛，易扭伤
- **治疗**：拉伸和灵活性训练

### 跟腱炎
- **原因**：过度使用、反复应力
- **症状**：跟骨处疼痛，尤其是早晨
- **治疗**：离心训练、拉伸、冰敷
